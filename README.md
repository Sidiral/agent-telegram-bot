## 📋 What is agent-telegram-bot?

agent-telegram-bot is a local AI assistant that works with Telegram on your Windows computer. It uses LM Studio and qwen-agent technology to help you do many tasks. You can search the web, analyze images, summarize web pages, access Wikipedia, do calculations, and manage different users with admin and guest roles. The bot works without needing to send your data online, keeping things local.

This bot supports multiple users on the same device. Admins can control access and settings, while guests have limited features. You interact with it directly through Telegram, making it easy to use without opening complex apps.
